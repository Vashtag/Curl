Normal Curling + Skill Draft MVP

How to play:
1) Open index.html in your browser.
2) Pick 1 of 3 pre-game skill cards.
3) Play a normal 8-end curling match vs AI.

Controls:
- Aim: mouse move or left/right arrows
- Charge throw: hold Enter or mouse down, release to throw
- Sweep: hold Space or mouse while your stone is moving
- Handle toggle: click Handle button

GitHub Pages deployment:
1) Push repo to GitHub
2) Settings -> Pages
3) Deploy from branch, choose your branch and /(root)
4) Save and open published URL

Display:
- The match UI now auto-fills the full browser window (fullscreen layout).
